# Data Modeling Demo (MongoDB + Mongoose)

This project is the lab scaffold for mastering MongoDB data modeling using **Mongoose schemas and models**.

## Goals
- Create **User** and **Task** Mongoose schemas
- Enforce **types**, **required fields**, and **defaults**
- Model the relationship using **reference-based association** (`ObjectId` + `populate()`)
- Implement basic **CRUD** routes

## Quick Start (once implementation is done)
1. Create a `.env` file:
   - `MONGODB_URI=your_mongodb_connection_string`
2. Install dependencies:
   - `npm install`
3. Run the server:
   - `npm run dev`
